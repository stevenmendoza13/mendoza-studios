INSTRUCCIONES - MENDOZA STUDIOS WEB V2

Esta versión tiene:
- Video de fondo reproduciéndose siempre.
- Espacio para logo.
- Dos botones principales:
  1. Contáctame y cuéntame tu idea
  2. Revisa mis trabajos realizados
- Sección de resultados comprobados.
- Espacio para imagen de evidencia de vistas.

CAMBIOS QUE DEBES HACER

1. Link del portafolio en Drive
Busca en index.html:

AQUI_PEGA_TU_LINK_DE_DRIVE

Reemplázalo por el enlace público de tu carpeta de Google Drive.

IMPORTANTE:
En Drive debes compartir la carpeta como:
"Cualquier persona con el enlace puede ver".

2. WhatsApp
Busca en index.html:

57TU_NUMERO_AQUI

Reemplázalo por tu número con código de país, sin espacios ni signos.

Ejemplo Colombia:
573003744851

Ejemplo Chile:
56912345678

3. Logo
Entra a la carpeta assets y coloca tu logo con este nombre exacto:

logo.png

Recomendación:
- Formato PNG.
- Fondo transparente si tienes.
- Tamaño recomendado: 500x500 px.

4. Video de fondo
Entra a la carpeta assets y coloca tu video de muestra con este nombre exacto:

video-muestra.mp4

Recomendación:
- Duración: 8 a 20 segundos.
- Formato: MP4.
- Peso ideal: menos de 20 MB.
- Sin audio.
- Que muestre varios ejemplos rápidos de tus trabajos.

5. Imagen de evidencia
Entra a la carpeta assets y coloca tu captura de resultados con este nombre exacto:

evidencia-vistas.jpg

Recomendación:
- Puede ser captura de YouTube Studio o analytics.
- Borra datos privados si aparecen.
- Usa una imagen clara donde se vean las vistas.
- Puede ser JPG o cambiar el nombre en el HTML si usas PNG.

6. Cómo publicarla gratis
Opción recomendada:
- Netlify.
- Arrastras la carpeta completa a Netlify y queda publicada.

También puedes usar:
- GitHub Pages.
